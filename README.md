# graduate-admission-prediction-machine-learning-app

## [Web App Demo](https://graduate-admission-prediction.herokuapp.com/)

**This is a Machine Learning web app to predict your chances of getting admission for higher studies after you provide the required information.**

The following are the fields on the basis of which prediction is made:
1. GRE Score
2. TOEFL Score
3. University Rating
4. Statement of Purpose Strength
5. Letter of Recommendation Strength
6. CGPA
7. Research Experience

## The Dataset
The dataset is taken from Kaggle - [Graduate Admission](https://www.kaggle.com/mohansacharya/graduate-admissions)
